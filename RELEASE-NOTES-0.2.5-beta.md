# Memory Governor 0.2.5-beta

This beta improves public positioning rather than the core contract.

## What Changed

- clarified that `memory-governor` is a governance kernel, not an execution-first productivity skill
- added a clearer “who it is for” section in the public README
- explained why the package feels heavier in small hosts and more valuable in complex multi-skill systems

## Why

External readers were correctly reading the package as policy-heavy and infrastructure-like.

This update makes that positioning explicit so the right users self-select earlier and smaller hosts do not install it with the wrong expectations.
