# Dev Notes

This directory is for maintainers, not normal skill installation.

这个目录给维护者使用，不是普通安装者的运行时必需内容。

It contains:

- `plans/`
  design notes, upgrade proposals, and evaluation plans
- `eval-lab/`
  governance evaluation scenarios for comparing competing designs

包含：

- `plans/`
  设计说明、升级提案、评估方案
- `eval-lab/`
  用来比较不同治理设计的评估场景

Normal users installing `memory-governor` do not need this directory to use the skill.

普通用户安装 `memory-governor` 时，不需要依赖这个目录才能使用 skill。
