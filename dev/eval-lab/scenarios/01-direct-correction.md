# Scenario 01: Direct Correction

## Setup

A user explicitly corrects the agent's tone:

- "不要写客服腔。"

This is clearly important, but it has only happened once.

## What To Evaluate

- Does the current design push this too quickly into `reusable_lessons`?
- Would an experimental `learning_candidates` layer hold it more safely?
- If held in a candidate layer, does that create value or just delay an obvious lesson?

## Desired Tension

This scenario is useful because both choices can look reasonable.
