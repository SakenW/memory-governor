# Publish

## English

This directory is for distribution artifacts, not development.

Use:

- `clawhub/`
  the publish-only skill directory for ClawHub

Refresh it with:

```sh
scripts/refresh-clawhub-package.sh
```

Then publish from:

```sh
publish/clawhub/
```
